Tools You Need: Zero Installation

Everything runs in the browser with Google Colab. Colab gives you a free notebook where text, code, charts, and results live together. Kaggle gives you free access to real datasets.

Open Colab.
Go to colab.research.google.com, sign in, and choose New Notebook.
Create your first cell.
Click the gray code cell and type print("Hello, Robot!").
Run a cell.
Press Shift+Enter. The output appears directly underneath the cell.
Create a free Kaggle account.
Go to kaggle.com. We use Kaggle because it hosts clean public datasets.
Get your Kaggle API key.
Kaggle account menu → Settings → API → Create New Token. It downloads kaggle.json.
Upload the key in Colab.
Each Kaggle project starts with an upload cell. Run it, choose kaggle.json, and the notebook downloads the dataset.

Kaggle key safety

Your kaggle.json key belongs to your account. Upload it only inside your own Colab notebook and do not paste it into public code.
