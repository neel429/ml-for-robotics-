Tools You Need: Zero Installation

Everything runs in the browser with Google Colab. Colab gives you a free notebook where text, code, charts, and results live together. Kaggle gives you free access to real datasets.

Open Colab.
Go to colab.research.google.com, sign in, and choose New Notebook.
Create your first cell.
Click the gray code cell and type print("Hello, Robot!").
Run a cell.
Press Shift+Enter. The output appears directly underneath the cell.
Create a free Kaggle account.
Go to kaggle.com. We use Kaggle because it hosts clean public datasets.
Get your Kaggle API key.
Kaggle account menu → Settings → API → Create New Token. It downloads kaggle.json.

Upload your Kaggle key to Colab

Every time you open a Colab notebook, it starts fresh like a brand new computer. That means you have to upload your kaggle.json key once per session before running any Kaggle download. Here is exactly how.

Open the project notebook.
Click the Open in Colab button in any project lesson. A new browser tab opens with the notebook.
Find the first code cell.
The very first cell in every project notebook is labelled Setup: Upload Kaggle Key. It looks like this:
Run the cell.
Press Shift+Enter or click the play button to the left of the cell. A file picker dialog opens directly in your browser tab.
Choose your kaggle.json file.
Navigate to wherever you saved kaggle.json, usually your Downloads folder. Select it and click Open. You will see a green confirmation message like kaggle.json(application/json) - X bytes, last modified: ... once it uploads successfully.
Run the next cell immediately.
The second cell moves your key to the location Kaggle expects: After this cell runs, your key is installed. Every !kaggle command in the rest of the notebook will work for this session.
⚠️
Colab sessions reset

Colab sessions reset when you close the tab or after 90 minutes of inactivity. You will need to upload kaggle.json again each time you reopen the notebook. This is normal.

💡
Keep the key private

Never paste the contents of kaggle.json directly into a code cell or share your notebook publicly while the key is present. The key is tied to your Kaggle account.
